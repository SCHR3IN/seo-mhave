# Задача M002: Сбор базового краула и карты индексируемых URL

## 🎯 Цель задачи
Повторно просканировать сайт, собрать полный реестр URL и разделить их по типам (разделы, товары, бренды, статьи, фильтры и др.). Для каждого адреса собрать метаданные (HTTP-код, canonical, robots/noindex, title, description, H1, og-теги) и выявить критические ошибки для дальнейшего исправления.

---

## 🛠️ Реализация и технические инструменты
Был разработан многопоточный Python-скрипт `seo_crawler.py`, который:
1. Подключается напрямую к базе данных 1С-Битрикс (MySQL) для извлечения активных URL (товары, категории, бренды, статьи, услуги).
2. Выгружает и сверяет адреса из файлов `sitemap.xml`.
3. Сканирует полученный список URL в 15 потоков с User-Agent `MHAVECrawler/1.0`.
4. Извлекает HTML-теги и анализирует дубликаты метатегов и заголовков.
