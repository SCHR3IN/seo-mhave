# Спецификация настройки SEO-фильтров («Аспро: Умный фильтр»)

Этот гайдлайн содержит требования и параметры настройки умного фильтра на сайте `mhave.ru` с использованием функционала готового решения Аспро. Цель — превратить страницы фильтрации в полноценные посадочные страницы, генерирующие трафик по низкочастотным (НЧ) и среднечастотным (СЧ) запросам.

## 2. Требования к ЧПУ (Человекопонятным Урлам) страниц фильтрации

По умолчанию Битрикс формирует некрасивые динамические ссылки вида `/catalog/.../?set_filter=y&arrFilter_123_456=y`.
Для SEO необходимо настроить ЧПУ-переходы (через настройки решения Аспро):
1. **Ссылка**: `/catalog/kosmetika/sredstva-dlya-litsa/filter/[код_свойства]-is-[код_значения]/apply/`
   * *Пример:* `/catalog/kosmetika/sredstva-dlya-litsa/filter/skin_type-is-dry/apply/` (для сухой кожи).
2. **Исключение из индексации (Canonical)**: Если выбрано более 2 параметров фильтрации одновременно (например, сухая кожа + акне + ретинол + Troiareuke), такая страница не должна индексироваться. Тег `canonical` должен вести на основную категорию или на наиболее частотную страницу фильтра (выбранную главным ориентиром).

## 3. Настройка метатегов и заголовков (SEO-правила)

Каждая посадочная страница SEO-фильтра должна иметь уникальные метатеги, генерируемые по шаблону или прописанные вручную в модуле «Умный фильтр» Аспро.

### 3.1. Шаблоны автогенерации
При выборе одного фильтра (например, `Ингредиент = Ретинол` в категории `Сыворотки`):
* **H1**: `[Название раздела] с [Свойство]` -> *Сыворотки для лица с ретинолом*
* **Title**: `Купить [H1] в интернет-магазине MHAVE с доставкой` -> *Купить сыворотки для лица с ретинолом в интернет-магазине MHAVE с доставкой*
* **Description**: `Большой выбор [H1] напрямую от производителей. Бесплатная доставка по Москве и РФ от 12000р. Закажите на MHAVE!`

### 3.2. Ручная привязка (для популярных страниц)
Для ВЧ/СЧ запросов (например, «крем для сухой кожи», «корейские кушоны») метатеги прописываются вручную маркетологом/SEO-специалистом в административной панели Битрикс:
1. Перейдите в: **Маркетинг -> Умный фильтр -> Условия**.
2. Создайте новое условие для нужного раздела каталога с выбором соответствующих значений свойств.
3. Задайте:
   - Собственный H1.
   - Мета-заголовки (Title, Description).
   - Блок уникального SEO-описания (выводится внизу страницы под листингом товаров).

## 4. Контроль качества (UX & SEO)
* **Товары**: Страница фильтрации не должна быть пустой. Если по комбинации свойств товаров нет в наличии, страница должна отдавать код ответа `444` (или `404`) либо закрываться тегом `noindex`.
* **Перелинковка**: Популярные теги (ссылки на SEO-фильтры) должны выводиться плиткой тегов (облаком тегов) над списком товаров для быстрого перехода пользователей и поисковых роботов.
