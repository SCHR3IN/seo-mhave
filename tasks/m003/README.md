# Задача M003: Сбор семантического ядра и кластеризация

## 🎯 Цель задачи
Собрать полное семантическое ядро сайта, обогатить частотностью Яндекс Wordstat, разделить фразы по сегментам и интентам, сгруппировать их в кластеры и сопоставить каждому кластеру наиболее релевантную посадочную страницу для исключения внутренней конкуренции (каннибализации) поисковых запросов.

---

## 🛠️ Реализация и технические инструменты
Был разработан интеллектуальный скрипт `semantic_core_builder.py`, выполняющий следующие шаги:
1. **Интеграция с API поисковых систем**:
   - Выгрузка реальных переходов из Яндекс Метрики (сохранены в `metrika_queries.json`).
   - Выгрузка показов и кликов из Яндекс Вебмастера (сохранены в `webmaster_queries.json`).
2. **Обогащение частотности (Wordstat)**:
   - Скрипт использует API Яндекс Директ (Wordstat) для парсинга точных частотников `"!фраза"`.
   - **Отказоустойчивость**: Реализован локальный кэш на базе SQLite (`WORDSTAT_LOCAL_DATABASE`) для обхода жестких лимитов API Яндекса (Rate Limits) и снижения стоимости запросов.
3. **Обработка текста и стемминг**:
   - Лексический анализ на основе стеммера Porter для русского и английского языков (очистка от стоп-слов, знаков препинания и дубликатов).
4. **Многомерная кластеризация**:
   - **Интент**: Коммерческий (Commercial), Гео (Geo), Информационный (Informational), Брендовый (Brand-Navigational).
   - **Сегмент**: Country-Origin, Brand, Problem-Skin, Active-Ingredient, Supplements.
5. **Алгоритм сопоставления URL**:
   - Скоринговое сопоставление фраз с метаданными из краула `url_registry.csv` на основе весов вхождений в Title, H1 и тело URL с приоритетом для БАДов и косметических брендов.
