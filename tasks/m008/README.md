# Задача M008: Подготовить ТЗ на SEO-шаблоны title, description и H1

## 🎯 Цель задачи
Разработать и согласовать детальное техническое задание (ТЗ) на автоматическую генерацию мета-тегов (`title`, `description`) и заголовков (`h1`) для всех типов страниц сайта `mhave.ru` с целью масштабного привлечения органического поискового трафика.

## 📌 Выполненные подзадачи

- [x] **M008/1 — Аудит типов страниц и анализ требований**: Сбор всех типов страниц, выявление дублей («Интернет-магазин», «Широкий выбор…») и определение интентов.
- [x] **M008/2 — Проектирование формул генерации и матрицы приоритетов**: Разработка шаблонов `title`, `meta description`, `H1` и зафиксирована иерархия приоритетов (Ручной ввод → SEO-свойства → Динамический шаблон → Fallback).
- [x] **M008/3 — Алгоритмизация краевых случаев и утверждение ТЗ**: Созданы алгоритмы склонений, вывода цен, обработки отсутствующих свойств и оформлен финальный документ [seo_templates_spec.md](seo_templates_spec.md).

---

## 🛠️ Реализация и технические изменения

### 1. Документ спецификации ТЗ
Подготовлен и согласован комплексный документ [seo_templates_spec.md](seo_templates_spec.md), регламентирующий правила генерации для 8 основных типов страниц:
* Главная страница
* Разделы каталога
* Подразделы каталога
* Страницы брендов
* Карточки товаров
* Результаты индексируемых фильтров (Smart Filter)
* Статьи и Блог
* Контентные и служебные разделы

### 2. Определение иерархии приоритетов
Сформулировано жесткое правило приоритета применения мета-тегов:
1. **Ручное значение** (заданное контент-менеджером на конкретной странице).
2. **SEO-свойства** инфоблоков/разделов.
3. **Шаблоны автоматической генерации** (динамический PHP-код).
4. **Fallback-значение по умолчанию** (название элемента/раздела).

### 3. Обработка краевых случаев
В ТЗ включены алгоритмы:
* Обработки товаров без указания объема или единицы измерения (исключение лишних пробелов и запятых).
* Склонения названий брендов и категорий.
* Корректного форматирования цен.
