# M008: Подготовить ТЗ на SEO-шаблоны title, description и H1

Подготовка детального технического задания (ТЗ) на генерацию мета-тегов (`title`, `description`) и заголовков `<h1>` для всех типов страниц интернет-магазина MHAVE.ru. 

## User Review Required

> [!IMPORTANT]
> ТЗ будет содержать формулы генерации для 8 основных типов страниц:
> 1. Главная страница
> 2. Разделы каталога (категории)
> 3. Подразделы каталога (подкатегории)
> 4. Страницы брендов
> 5. Карточки товаров
> 6. Результаты индексируемых фильтров (Smart Filter)
> 7. Статьи/Блог
> 8. Контентные/служебные страницы
>
> Также в документе будет жестко закреплена иерархия приоритетов:
> **Ручное значение** (свойства страницы/элемента) → **SEO-свойства инфоблока** (наследуемые мета-теги Битрикс) → **Шаблон генерации** → **Дефолтный Fallback**.
