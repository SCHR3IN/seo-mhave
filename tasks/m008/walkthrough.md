# Walkthrough: ID M008 — Подготовить ТЗ на SEO-шаблоны title, description и H1

Задача по подготовке технического задания (ТЗ) на SEO-шаблоны генерации мета-тегов и заголовков успешно завершена.

## Выполненная работа

### Документ спецификации ТЗ
Создан подробный файл спецификации:
* [seo_templates_spec.md](file:///home/egor/Документы/Antigravity_Project/mhave.ru/seo/tasks/m008/seo_templates_spec.md)

Документ описывает:
1. **Приоритеты применения мета-тегов**: иерархия от ручного заполнения через наследуемые свойства Битрикса к динамическому PHP-шаблону и дефолтному Fallback.
2. **Адаптивные формулы генерации** для 8 типов страниц:
   - Главная страница
   - Разделы каталога
   - Подразделы каталога
   - Страницы брендов
   - Карточки товаров (с обработкой отсутствия объема, единиц измерения и цены)
   - Результаты индексируемых фильтров (Smart Filter)
   - Статьи и Блог
   - Контентные и служебные страницы (О компании, Доставка и т.д.)
3. **Логические ограничения для разработчиков**: проверка на пустоту строк, очистка цен, склонение топонимов, лимиты длины (Title < 80, Description < 180).

## Валидация

Все формулы и примеры были перепроверены на соответствие следующим SEO-требованиям:
- Наличие коммерческих интентов (слова *«купить»*, *«цена»*).
- Региональная привязка (фразы *«в Москве»*, *«с доставкой по России»*).
- Присутствие названия интернет-магазина *«MHAVE»* в конце тега Title.
- Отсутствие спам-конструкций в заголовках `<h1>`.
