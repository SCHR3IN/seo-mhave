# Задача M006: Стабилизировать дочерние sitemap

## 🎯 Цель задачи
Обеспечить стабильную, быструю и гарантированную отдачу дочерних XML-файлов карты сайта (`sitemap-files.xml`, `sitemap-iblock-9.xml` и др.) без превышения времени ожидания (timeout). Предотвратить сбои при сканировании роботами Яндекса и Google.

---

## 🛠️ Реализация и технические изменения

### 1. Отдача Sitemap напрямую через Nginx
Основной причиной таймаутов была обработка запросов к статическим XML-файлам через веб-сервер Apache и интерпретатор PHP, что приводило к очередям и задержкам при высокой нагрузке. 

Мы вынесли обработку всех файлов карты сайта на уровень Nginx. Создан файл конфигурации `/etc/nginx/bx/site_settings/mhave.ru/sitemaps.conf` со следующими правилами:
```nginx
location ~* ^/sitemap.*\.xml$ {
    root /home/bitrix/ext_www/mhave.ru;
    expires 24h;
    add_header Content-Type "application/xml; charset=utf-8";
    add_header X-Static-Source "Nginx" always;
    access_log off;
}
```
Благодаря этому Nginx отдает файлы напрямую с диска, минуя бэкенд.

### 2. Применение конфигурации и проверка
* Проведена валидация синтаксиса Nginx (`nginx -t`).
* Перезапущена служба Nginx (`systemctl reload nginx`) на сервере `147.45.143.60`.
* Протестирован генератор карты сайта и скрипт постобработки.
