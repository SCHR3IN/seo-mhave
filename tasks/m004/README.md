# Задача M004: Оптимизация Sitemap и автоматизация очистки URL

## 🎯 Цель задачи
Оптимизировать файл `sitemap.xml` для `mhave.ru`, очистив его от «мусорных» URL (параметры SKU `?oid=`, служебные страницы `/auth/`, `/basket/`, мягкие ошибки 404 `/contacts/stores/moskva/404/` и битые страницы). Настроить автоматический запуск генерации и постобработки sitemap на сервере, чтобы в индекс попадали только качественные canonical-страницы с кодом 200.

---

## 🛠️ Реализация и технические изменения

### 1. Отключение SKU (Iblock 15) в настройках Битрикс
Создан и запущен скрипт [update_settings.php](update_settings.php), который через Bitrix ORM изменил конфигурацию карты сайта ID 1:
```php
$settings['IBLOCK_ACTIVE'][15] = 'N';
$settings['IBLOCK_ELEMENT'][15] = 'N';
```
Это предотвращает изначальное попадание торговых предложений (SKU) в генерируемые XML-файлы.

### 2. Синхронный запуск генерации Битрикс
Создан скрипт [run_generation.php](run_generation.php), который запускает пошаговую генерацию sitemap через CLI:
```php
$generator = new \Bitrix\Seo\Sitemap\Generator(1);
while ($generator->getStep() < Step::STEPS[Step::STEP_INDEX]) {
    $generator->run();
    $generator->init($generator->getStep(), $generator->getState());
}
```

### 3. Скрипт очистки и валидации
Разработан многопоточный Python-скрипт [clean_sitemap.py](clean_sitemap.py), который:
* Считывает все сгенерированные Битриксом суб-sitemaps (`sitemap-*.xml`).
* Фильтрует адреса по правилам `robots.txt`.
* Проверяет каждый URL на HTTP-код 200.
* Парсит HTML каждой страницы и исключает адреса с мета-тегом `noindex` или несовпадающим тегом `rel="canonical"`.
* Объединяет все валидные URL в единый плоский оптимизированный `sitemap.xml`.

### 4. Автоматизация через Cron
Создан баш-скрипт [sitemap_cron.sh](sitemap_cron.sh), который запускает последовательно генерацию и очистку с логированием в `sitemap_cron.log`.
Скрипт добавлен в системный планировщик `/etc/cron.d/bx_dbmhave` для ежедневного выполнения в **03:00**:
```cron
0 3 * * * bitrix /bin/bash /home/bitrix/ext_www/mhave.ru/seo/tasks/m004/sitemap_cron.sh >/dev/null 2>&1
```
