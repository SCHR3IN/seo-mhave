# M007: Добавить единственный H1 на главную страницу

Добавить на главную страницу один видимый заголовок первого уровня `<h1>`, отражающий ассортимент и коммерческий интент. Согласно рекомендациям аудита, рабочая формулировка заголовка: «Профессиональная косметика, космецевтика и пищевые добавки Mhave»; проверить отсутствие второго H1.

## User Review Required

> [!IMPORTANT]
> Для соблюдения дизайн-системы и визуальной гармонии заголовок `<h1>` будет интегрирован в блок `CUSTOM_TEXT` (блок иконок категорий), расположенный сразу под главным баннером. Ему будут присвоены стандартные классы темы `index-block__title switcher-title` для автоматического наследования адаптивных стилей шрифтов (размеры, отступы на desktop/mobile).
