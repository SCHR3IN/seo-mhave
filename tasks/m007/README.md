# Задача M007: Добавить единственный H1 на главную страницу

## 🎯 Цель задачи
Обеспечить правильную семантическую структуру главной страницы путем внедрения единственного заголовка первого уровня `<h1>`. Заголовок должен быть оптимизирован под поисковые запросы, не дублировать мета-тег `<title>` и лаконично вписываться в общий дизайн сайта на десктопной и мобильной версиях.

---

## 🛠️ Реализация и технические изменения

### 1. Выбор места размещения заголовка H1
Первоначальный вариант размещения H1 в верхнем блоке категорий (`CUSTOM_TEXT`) был отклонен для разгрузки первого экрана. Совместно с заказчиком выбрано более сбалансированное решение — размещение `<h1>` в нижней части страницы, над основным SEO-текстом компании (в блоке «О компании» / `front_company`).

### 2. Разработка и внедрение шаблона
Изменен шаблон компонента детального описания компании `/local/templates/mhave/components/bitrix/news.detail/front_company/template.php` на боевом сервере (создана резервная копия `.bak`):
* Вместо стандартного вызова функции вывода `<h2>` встроен явный блок с тегом `<h1>`, использующий оригинальные стили оформления темы (`index-block__title switcher-title`).
* Переопределена переменная названия блока на оптимизированную ключевую фразу:
  ```php
  $arParams['TITLE'] = 'Интернет-магазин профессиональной косметики, космецевтики и пищевых добавок Mhave';
  ```
