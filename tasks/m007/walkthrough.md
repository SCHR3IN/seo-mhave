# Walkthrough: ID M007 — Добавить единственный H1 на главную страницу

Задача по добавлению единственного H1 на главную страницу успешно завершена с учетом пожеланий по размещению в нижней части страницы.

## Изменения

### 1. Шаблон блока категорий `CUSTOM_TEXT`
Файл `/home/bitrix/ext_www/mhave.ru/include/mainpage/components/custom_text/text.php` возвращен к исходному состоянию (без заголовка `<h1>` перед списком иконок категорий).

### 2. Шаблон блока «О компании» (`front_company`)
Модифицирован шаблон `/home/bitrix/ext_www/mhave.ru/local/templates/mhave/components/bitrix/news.detail/front_company/template.php` на боевом сервере (предварительно сделана резервная копия `.bak`).
- Вывод заголовка через `showTitleInLeftBlock` (который выводит `<h2>`) заменен на вывод `<h1>` с сохранением оригинальных стилей (`index-block__title switcher-title`) и сетки верстки.
- Переопределен текст заголовка на более релевантный и целевой для SEO (не дублирующий `<title>` страницы):
  ```php
  $arParams['TITLE'] = 'Интернет-магазин профессиональной косметики, космецевтики и пищевых добавок Mhave';
  ```

## Тестирование и валидация

### 1. Проверка HTML-кода
Сделан проверочный запрос на главную страницу. Результаты парсинга подтверждают наличие ровно одного тега `<h1>` на странице:
```bash
$ curl -s https://mhave.ru/ | grep -E -o -i "<h1[^>]*>" | wc -l
1
```

Содержимое тега корректно:
```html
<h1 class="index-block__title switcher-title">Интернет-магазин профессиональной косметики, космецевтики и пищевых добавок Mhave</h1>
```
